AI 02

Gen-AI Model Building

In this Generative AI Competition, participants will create a website using AI tools like ChatGPT or Gemini. They will generate HTML,CSS, and JavaScript code through prompts, build the website using an online platform and publish it. Participants must submit both the prompt used and the live website link through the Google Form. The competition focuses on creativity, prompt-writing skills, and building a visually appealing, user-friendly website.
