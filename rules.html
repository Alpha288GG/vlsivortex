Regulations

Rules &
Guidelines

Please read all rules carefully. All decisions by judges are final and binding.

🛡️ General Event Guidelines
↓
  • ▸ID Requirements: All participants must carry valid college ID and event badge at all times.
  • ▸Punctuality: Report to event venues at least 30 minutes before scheduled start. Late arrivals may be disqualified.
  • ▸Mobile Phones: Prohibited during competition hours unless explicitly permitted.
  • ▸Decorum: Misbehavior, arguing with judges, or property damage leads to immediate disqualification.
  • ▸Judges' Decision: All judges' decisions are final. No appeals will be entertained.
💻 Coding Competition Rules
↓
  • ▸Platform: Closed-network judge. Languages: C, C++, Java, Python 3.
  • ▸Internet: Strictly blocked. Offline documentation provided.
  • ▸Scoring: Points per test case passed. Ties broken by completion time.
  • ▸Plagiarism: Automated similarity tools active. Copying results in an instant ban.
🤖 Gen-AI Model Building Rules
↓
  • ▸Problem Statement: Released exactly at the event start time.
  • ▸Infrastructure: Use only provided cloud compute. No personal proprietary API keys (GPT-4 etc.).
  • ▸Models: Open-source only — Llama, Mistral, etc.
  • ▸Judging: Innovation 30%, Performance 40%, Scalability 15%, Presentation 15%.
⚡ Circuit Making / Model Designing Rules
↓
  • ▸Components: Standard kit provided. External components not allowed.
  • ▸Safety: Strict lab safety rules apply. Intentional short-circuiting leads to expulsion.
  • ▸Deliverables: Working prototype + hand-drawn schematic within time limit.
